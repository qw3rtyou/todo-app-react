## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Build for production: `npm run build`

## Docker Support

Run with Docker:

```bash
docker-compose up --build
```

Access the application at http://localhost:5555
